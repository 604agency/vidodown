# Vido Down — 604 Agency

Video indirme web uygulaması. YouTube, Instagram, TikTok ve 1000+ siteyi destekler.

## Kurulum (Railway)

1. Bu repoyu GitHub'a yükle
2. railway.app adresine git, GitHub ile giriş yap
3. "New Project" → "Deploy from GitHub repo" → bu repoyu seç
4. Otomatik deploy olur, link verilir!

## Yerel Çalıştırma

```bash
pip install -r requirements.txt
python app.py
```
